# Эцсийн хэл найруулгын шалгалтын жагсаалт

Глобал `mongolian-academic` skill-ийн дагуу бичвэр бичигдсэний ДАРАА шалгах:

## 1. Эгшгийн зохицол (Vowel harmony)

### Эрэгтэй (а, о, у, я) → **-ын / -ыг / -аас**
- ✅ судалгаа**ны** үр дүн
- ✅ туршилт**ын** тайлан
- ✅ өгөгдл**ийг** (и саармаг → тухайн үгийн гол эгшгийг харна: ө → -ийг)

### Эмэгтэй (э, ө, ү, е) → **-ийн / -ийг / -ээс**
- ✅ хөдөлмөр**ийн** орлого
- ✅ боловсрол**ын** (о → -ын) — анхаар!
- ✅ дүгнэлт**ийн**
- ✅ өгөгдл**ийн**

### Түгээмэл алдаанууд шалгах үгс
| Буруу | Зөв |
|---|---|
| сургууль**н** | сургууль**ийн** |
| судалгаа**н** үр дүн | судалгаа**ны** үр дүн |
| эконометрик**н** | эконометрик**ийн** |
| босго**н** | босго**ны** (о → -ны) |
| ажилч**ы**н | ажилч**ин**ы хэрэв үнэн эзэмшил, эсвэл "ажилтнууд"-ын |

## 2. Давхар эгшиг

- ✅ **аа**: цалингаас (цалнгаас биш), олж авна, хараагаар
- ✅ **ээ**: хэрэгслээр, эдээр
- ✅ **оо**: өгөөж (өгөж биш!), тогтоогдсон, илэрлээ
- ✅ **өө**: өөрийн, тухайгаа
- ✅ **уу**: хуучин
- ✅ **үү**: үүгэл, дүүрэг, үүсэх
- ✅ **ии**: ихээхэн, дийлэнхи

## 3. " " ишлэл тэмдэг (СЭЗИС стандарт)

❌ «текст» (Монгол тэмдэглэл — СЭЗИС шаардахгүй)  
✅ "текст" (шулуун хос хашилт — СЭЗИС стандарт)

## 4. Тоо

- ✅ "нэг, хоёр, гурав... ес" → **үсгээр** (1-9)
- ✅ "10, 50, 100" → **тоогоор** (10+)
- ✅ Өгүүлбэрийн ЭХЭНД: "Тавин судалгаа..." (50 биш)
- ✅ Хэмжээстэй тоо бүгд тоогоор: "5 жил", "25%", "12,020 хүн"
- ✅ Мянгыг тусгаарлах: "49,366" эсвэл "49 366" — бүхэл бичвэрт НЭГ стандарт

## 5. Зураас (СЭЗИС шинэчлэгдсэн)

- **— (em dash): АШИГЛАХГҮЙ** — user-аас тодотгосон
- **–** (en dash): интервал дахь зай (2020–2024 он, х. 45–67)
- **-** (hyphen): нийлмэл үг (IV-ийн, 2SLS-ээр)

## 6. "Би", "бид"-ийг ХАСАХ

| Буруу | Зөв |
|---|---|
| "Бид энэ загварыг ашигласан" | "Энэ загварыг ашиглан..." / "Загварыг хэрэглэсэн" |
| "Би эндээс харж байна" | "Үүнээс харахад..." / "Ажиглагдаж байна" |
| "Бидний үр дүн..." | "Шинжилгээний үр дүн..." / "Эмпирик үр дүн..." |

## 7. "-лаа/-лоо" төгсгөл → идэвхгүй хэлбэр

| Ярианы | Академик |
|---|---|
| "загварыг гаргалаа" | "загварыг гаргав" / "загвар гаргагдсан" |
| "тооцооллоо" | "тооцсон" / "тооцоо хийгдсэн" |
| "үзүүлэв" | "харуулав" / "илэрсэн" |

## 8. Нэр томьёоны тогтвортой байдал

Нэг ойлголт = нэг нэр (бүхэл бичвэрт):

- **"IV"** vs "хэрэгсэл хувьсагч" — эхний удаа тодорхойлоод тогтмол нэгийг
- **"threshold"** vs "босго" — Монголоор бичнэ
- **"HSES"** vs "ӨНЭЗС" — Монголоор ашиглаж, англи хувилбарыг хаалтанд
- **"regime"** — "дэглэм" эсвэл "хэсэг" — нэгийг сонго
- **"endogeneity"** = "эндогень хамаарал"
- **"exclusion restriction"** = "хасагдах нөхцөл"

## 9. Хүснэгт, Зурагын тайлбар (СЭЗИС стандарт)

### Хүснэгт — ДЭЭР (дугаар + Italic нэр)
```
Хүснэгт 1.1  Шинжилгээний түүврийн үндсэн үзүүлэлт
             [Italic]
```
- Дугаар бүлгээр (1.1, 1.2, 2.1...)
- Зүүн дээд булан
- Доор нь эх сурвалж: 10pt Italic

### Зураг — ДООР (дугаар + Italic нэр)
```
Зураг 3.1  Босготой регрессийн grid search үр дүн
           [Italic]
Эх сурвалж: ӨНЭЗС 2020-2024, зохиогчийн тооцоо.
            [10pt, Italic]
```
- Дугаар бүлгээр
- 300 dpi нягтралтай
- Голд байрлуулна

### Текстэд заавал ишлэх (" " ишлэлтэйгээр)
- ✅ "Хүснэгт 2.1-ээс үзэхэд ..."
- ✅ "Зураг 3.1-д харуулсанчлан ..."
- ❌ Хүснэгт/зургийг текстэд огт дурьдахгүй орхих

## 13. Гадаад үгсийн буулгалт

- **computer** → **компьютер** (тогтсон)
- **internet** → **интернэт**
- **regression** → **регресс** (загварт)
- **instrument** → **хэрэгсэл** (Монголоор) эсвэл **инструмент** (контекстээс)
- **endogeneity** → **эндогень хамаарал** (эдийн засгийн стандарт)
- **coefficient** → **коэффициент**
- **microdata** → **микро өгөгдөл**

## 14. "Тогтоосон" vs "тогтоогдсон"

- **Тогтоосон** = идэвхтэй, хэн нэгэн тогтоосон ("Судлаач ... тогтоосон")
- **Тогтоогдсон** = идэвхгүй, автоматаар ("... тогтоогдсон")
- Академикт **идэвхгүй** хэлбэр давамгайлна: "тогтоогдсон", "тодорхойлогдсон", "илэрсэн"

## 15. Үзүүлэлт vs Үзүүлэх

- **Үзүүлэлт** = тоон өгөгдөл (indicator)
- **Үзүүлэх** = харуулах (to show)

## Эцсийн шалгалт (жишээ өгүүлбэр)

### Муу хувилбар
> "Бид энэ судалгаандаа боловсролын өгөөжийн тухай IV threshold регрессийг ашиглалаа. Үр дүн нь маш сонирхолтой байсан."

### Зөв хувилбар
> «Энэхүү судалгаанд боловсролын өгөөжийг хэмжих зорилгоор хэрэгсэл хувьсагчийн босготой (IV-Threshold) регрессийн аргыг хэрэглэв. Шинжилгээний үр дүнгээс үзэхэд боловсролын жилийн 13 цэг дээр статистик ач холбогдолтой бүтцийн өөрчлөлт илэрсэн (p < 0.001).»

**Ялгаа:**
- «» (ишлэл)
- "бид" → идэвхгүй хэлбэр
- "ашиглалаа" → "хэрэглэв"
- "маш сонирхолтой" → тоон баримттай ("p < 0.001")
- Нэр томьёо бүрэн: "хэрэгсэл хувьсагчийн босготой (IV-Threshold) регрессийн арга"
