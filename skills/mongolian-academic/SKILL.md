# Монгол академик бичлэгийн ур чадвар (СЭЗИС стандарт)

Энэ skill нь **СЭЗИС-ийн "Оюутны бие даан гүйцэтгэх ажил, тайлан бичих
ерөнхий шаардлага, формат"** стандартын дагуу Монгол эрдэм шинжилгээний
бичлэгийн дүрмийг тодорхойлно.

> **Эх сурвалж:** `ОЮУТНЫ-БИЕ-ДААН-ГҮЙЦЭТГЭХ-АЖИЛ.pdf` (12 хуудас). Татсан
> текстийн хувилбарыг `inputs/sezis_format_extracted.md`-ээс үзэж болно.
>
> **Зэрэгцсэн skill-үүд:**
> - `SEZIS_FORMAT.md` — дэлгэрэнгүй нарийн дүрэмт
> - `CHECKLIST.md` — эцсийн шалгалтын жагсаалт
> - Глобал `~/.claude/skills/mongolian-academic/` — Монгол хэлний зөв бичиг

## I. Цаас, захын зай (PDF 3.13)

- **Цаас:** A4 (210 × 297 mm)
- **Margins:**
  - Top: **25 мм** (2.5 см)
  - Bottom: **20 мм** (2 см)
  - Left: **30 мм** (3 см, үдэх зай багтсан)
  - Right: **20 мм** (2 см)

## II. Үндсэн бичвэр (PDF 3.7)

- **Фонт:** Times New Roman **12pt**
- **Justification:** Баруун зүүн зэрэгцсэн (Justified)
- **Line spacing:** Multiple At **1.5sp**
- **Paragraph spacing:** **After 6pt, Before 6pt**
- **Өргөтгөл:** `.doc` + `.pdf` (PDF 3.6)

## III. Хуудасны дугаарлалт (PDF 3.8)

- **Байрлал:** **Баруун дээд булан** (top-right)
- **Формат:** Араб тоо (1, 2, 3 ...)
- **Эхлэл:** Оршил хэсгээс эхлэн
- **Хавсралт:** Үргэлжлүүлэн (21, 22, 23...), нийт хуудасны тоонд
  **орохгүй** (PDF 3.12)

## V. Ишлэлийн тэмдэг (СЭЗИС шаардлага)

- ✅ **" "** (шулуун хос хашилт, straight double quotes)
- ❌ **« »** (guillemet) — ашиглахгүй
- ❌ **— (em dash)** — ашиглахгүй; оронд `–` (en dash, интервал) ба `-`
  (hyphen, нийлмэл үг)

## VI. Хүснэгтийн формат (PDF 3.15)

### Дугаар ба нэр — **ХҮСНЭГТИЙН ДЭЭД ТАЛД, ЗҮҮН ДЭЭД БУЛАН**
```
Хүснэгт 1.1  Шинжилгээний түүврийн үндсэн үзүүлэлт
             [Italic]
```
- **Дугаар бүлгээр:** Хүснэгт 1.1, 1.2, 2.1...
- **Нэр:** Дугаарын ард Italic
- **Font:** 10pt

### Эх сурвалж — **ХҮСНЭГТИЙН ДООД ТАЛД**
```
Эх сурвалж: ӨНЭЗС 2024, зохиогчийн тооцоолол
            [10pt, Italic]
```

### Агуулга
- **Spacing:** After 2pt, Before 2pt (PDF 3.15.5)
- Үргэлжилсэн хуудсанд: «**Үргэлжлэл Хүснэгт 1.1**» + толгой гарчиг давтана

## VII. Зургийн формат (PDF 3.16)

### Дугаар ба нэр — **ЗУРГИЙН ДЭЭД ТАЛД, ЗҮҮН ДЭЭД БУЛАН** ⚠️
```
Зураг 1.2  Цалин ба боловсролын жилийн хамаарал
           [10pt, Italic]
     [--- зураг ---]
Эх сурвалж: ӨНЭЗС 2024 [10pt, Italic]
```

> ⚠️ **АНХААР:** Миний өмнөх skill хувилбар "зургийн доор" гэж буруу
> тодотгосон. PDF 3.16.2-ын дагуу зургийн нэр **зүүн дээд** буланд
> (зургийн ДЭЭР) байрлана.

- **Нягтрал:** 300 dpi (PDF 3.16.5)
- **Бүдүүвч, диаграмм, график, фото** — бүгд "Зураг" нэрээр нэгдсэн
  дугаарлалтад (PDF 3.16.1)

## VIII. Математик томьёо (PDF 3.16 доод хэсэг)

- **Word Equation** ашиглана (Pandoc-д OMML автомат)
- **Дугаар:** Араб тоо, **[ ] хаалтанд**, хуудасны **баруун захад**
  нэг мөрөнд
- **Тайлбар:** «Энд:» гэж эхлээд бүх тэмдэглэлийг нэрлэнэ
  (алгебрийн хүртвэр эхэлж, дараа хуваарь)

```
ln w_i = α + β · educ_i + γ · X_i + ε_i                              [1]

Энд:
    ln w_i — i-р хүний сарын цалингийн логарифм
    educ_i — i-р хүний боловсролын жил
    ε_i    — алдааны гишүүн
```

## IX. Ном зүй (PDF 8-9 хуудас)

- **Гарчиг:** `НОМ ЗҮЙ` [**ALL CAPS**, **Bold**, **голд (center)**]
- **Байрлал:** Үндсэн бичвэрийн дараа, хавсралтын өмнө, **шинэ хуудас**
- **Font:** Үндсэн бичвэртэй адил (TNR 12pt)
- **Line spacing:** Multiple At 1.5sp
- **Paragraph spacing:** After 0pt, Before 0pt
- **Дугаарлалт:** Араб тоо, нэг баганад зэрэгцсэн (1., 2., 3.)
- **Bold БИШ** ердийн бичвэрээр
- **Hanging indent:** Зохиогчийн нэр эхний мөрнөөс, дараагийн мөрүүд
  догол мөрнөөс
- **APA** стандарт
- Жич: «Эш татах, эх сурвалжийг заах, ном зүйн бичилт хийх стандарт»-ыг
  мөрдөнө

## X. Хавсралт (PDF 3.9-3.12)

- **Ганц хавсралт:** Зүгээр "Хавсралт" [Bold]
- **Олон хавсралт:** Хавсралт A, B, C... [Bold]
- Хавсралт тус бүрийг шинэ хуудаснаас
- Хуудасны дугаар үндсэн бичвэрээс үргэлжлэнэ, **нийт хуудасны тоонд
  орохгүй**

## XI. Жагсаалт (PDF 3.14)

- **Bullet:** цэгэн тэмдэг **•** (Paragraph → Bullets → "•")
- Бусад тэмдэгт (○, ●, ◆) ашиглахгүй
