# Claude Code-д өгөх жишээ prompt-ууд

Сайн prompt = сайн тайлан. Энд янз бүрийн нөхцөлд ашиглах prompt жишээнүүд.

## Энгийн курсын ажил

```
Сэдэв: "Монголын МУГ-ын ИБП (Хүний хөгжлийн индекс)-ийн 
өөрчлөлт 2000-2023 он"

Шаардлага:
- 15-20 хуудас
- Бакалаврын 3-р дамжааны түвшний курсын ажил
- inputs/ доторх PDF, Excel-г эх сурвалж болгон ашигла
- inputs/hdi-chart.png зургийг хэрэглэ
- APA стандартаар иш тат
- МУИС-ийн эдийн засгийн тэнхимийн стандартаар

Эхлээд төлөвлөгөө гаргаж надаас баталгаажуул.
```

## Дипломын ажил

```
Бакалаврын дипломын ажил:
Сэдэв: "Ковид-19 дараах Монголын аялал жуулчлалын салбарын 
сэргэлтийн шинжилгээ"

Шаардлага:
- 50-60 хуудас
- 4 бүлэг: Онол | Арга зүй | Судалгаа | Дүгнэлт
- inputs/ дотор 8 файл бий — шаардлагатайг нь ашигла
- Ядаж 5 хүснэгт, 8 зураг/график
- 30-аас доошгүй эх сурвалж
- Өөрийн судалгаа (survey) хийсэн, inputs/survey-results.xlsx-д үр дүн

Эхлээд бүтцийн төлөвлөгөө + гол гол тоо баримтын жагсаалт гарга.
```

## Лабораторийн тайлан

```
Химийн лабораторийн тайлан бич:
- Туршилт: "Хүчиллэг ба шүлтлэг уусмалын pH хэмжилт"
- inputs/lab-data.xlsx-д туршилтын өгөгдөл
- inputs/ph-meter.jpg багажны зураг

Стандарт бүтэц: Зорилго → Онол → Багаж хэрэгсэл → Арга зүй 
→ Үр дүн → Дүн шинжилгээ → Дүгнэлт

5-7 хуудас, ШУТИС-ийн стандартаар.
```

## Товч эрдэм шинжилгээний өгүүлэл

```
Эрдэм шинжилгээний өгүүлэл бич, IMRaD (Introduction, 
Methods, Results, Discussion) бүтэцтэй.

Сэдэв: "Улаанбаатар хотын агаарын бохирдлын төвлөрлийн 
улирлын хэлбэлзэл"

- 10-12 хуудас
- inputs/air-quality-data-2020-2024.xlsx эх сурвалж
- Хураангуй (Abstract) 150 үгтэй, түлхүүр үгтэй
- Зүүлт (footnote) биш, APA in-text citation
- "Монголын байгаль орчны сэтгүүл"-д нийтлэх түвшний
```

## Тайланг засах, сайжруулах

### Агуулгыг нэмэх
```
outputs/tailan.docx-ийг унш.

2-р бүлэгт дараах агуулгыг нэм:
- Хятадын жишээ туршлагын хэсэг (1.5 хуудас)
- Өмнөх судлаачдын дүгнэлттэй харьцуулсан хэсэг

inputs/china-case-study.pdf-ээс иш татна уу.
outputs/tailan-v2.docx нэрээр хадгал.
```

### Форматыг засах
```
outputs/tailan.docx-ийг validate.py-аар шалгаад алдаа 
гарсан бол зас.

Бас:
- Хүснэгт 3, 4, 5-ийн багана өргөнийг адилхан болго
- Зураг 2-ын доор эх сурвалж байхгүй байна, "Эх сурвалж: 
  ӨССАЙ, 2023" гэж нэм
- 15-р хуудас дээрх зай алдагдсаныг шалгаж зас
```

### Хэл найруулгыг сайжруулах
```
outputs/tailan.docx-ийг бүхэлд нь унш.

Дараах алдаануудыг шалгаж зас:
1. Эгшгийн зохицолын алдаа (-ын vs -ийн)
2. Давхар эгшиг дутуу бичсэн үг (аа, ээ, оо, өө, уу, үү)
3. Эхний биет (би, бид) хэрэглэсэн өгүүлбэрийг идэвхгүй 
   хэлбэрт хөрвүүл
4. Ярианы өнгөтэй үг илэрсэн бол академик өнгөтэй болгож 
   орлуул
5. Нэр томьёоны зөрчил байвал тогтвортой болгож нэгтгэ

Засвар хийсэн хэсэг бүрийг товч жагсаан харуул.
```

### Тусгайлсан хэсэг бичих
```
Тайлангийн 3-р бүлгийн 3.2 дэд бүлэгт "Статистик шинжилгээ" 
гэсэн хэсэг нэм.

- 3-4 хуудас
- inputs/survey-results.xlsx-ийн өгөгдөл дээр үндэслэсэн 
  тодорхойлолт
- Дор хаяж 2 график (bar chart + pie chart)
- chi-square тестийн үр дүнг тайлбарласан
- Тайлбар нь статистик мэдэхгүй уншигчид ойлгомжтой
```

## Зөвлөмж: сайн prompt бичих дүрэм

### 1. Нэг юм асуу
Муу: "Тайлан бичиж өг"
Сайн: "15 хуудас курсын ажил, МУИС-ийн стандартаар, inputs/ 
доторх 3 файл ашиглан"

### 2. Хязгаарыг тогтоо
- Хэдэн хуудас?
- Ямар бүтэц?
- Хэдэн эх сурвалж?
- Хэдэн хүснэгт/зураг?

### 3. Жишээ өг
"МУИС-ийн бакалаврын дипломын ажлын стандарт" гэхээс "иймэрхүү 
бүтэцтэй: inputs/sample-thesis.pdf" гэвэл илүү тодорхой.

### 4. Баталгаажуулалт шаард
Prompt-ийн эцэст "төлөвлөгөөг харуулж надаас баталгаажуул" гэж 
бич. Алдаа гаргахаас хамгаална.

### 5. Чанарын шалгалт хүс
"Хийж дууссаны дараа validate.py-аар шалга, үр дүнг мэдээл" 
гэж заавал нэм.
